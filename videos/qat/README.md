# QAT Training Videos

Place QAT (Quantum Alignment Technique) training videos here.

**Supported formats:** MP4, MOV, MKV, M4V, WEBM, AVI

**Transcription:**
```bash
python scripts/transcribe_videos.py --type qat
python scripts/transcribe_videos.py --type qat --ingest   # also ingest into Qdrant
```

**Content type:** `training_qat`
**Qdrant collection:** `training_materials`

Note: QAT content is for internal clinical use only (`internal_only: true`).
QAT "plaatjes" (visualization images) are never mentioned in public-facing content.
