# Video Transcripts

Generated by `scripts/transcribe_videos.py`. One pair of files per video:

- `{video_stem}.json` — full transcript with timestamps per segment:
  ```json
  {
    "source_file": "NRT_Shoulder_Advanced.mp4",
    "transcribed_at": "2026-04-14T15:00:00Z",
    "segments": [
      {"start": 0.0, "end": 4.2, "text": "Welkom bij de NRT schouder training."},
      ...
    ]
  }
  ```

- `{video_stem}.txt` — plain text for easy reading and manual editing

This directory is gitignored (transcripts can be large).
Re-run `transcribe_videos.py` to regenerate.
